# TaskFlow : A Todo App

## Features :

### User can :

- Create todo
- Update todo
- Mark as completed
- Delete Todo
- Filtering based on priority and completed or not
- User can search based on keyword, multiple keyword also works

## Technology Used :

- React Js
- TailwindCss
- React Hook From
- Zod
- React Hot Toast
- TypeScript
